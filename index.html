DirecTree: Directory Tree Viewer 🌳

   

Visualize your local directory structure interactively on the web.

   

Step 1: Download the Python script make_tree.py.

Step 2: Run the script from your terminal to generate a tree.json file.

python make_tree.py /path/to/your/directory

This will create a tree.json file in the same directory where you ran the script, representing your directory structure.

   
   

2. Interactive Tree Viewer 🌲

Upload the tree.json file you just generated to view it as a collapsible directory tree.
